# LM Studio

[LM Studio](https://lmstudio.ai/) allows you to run local LLMs with an OpenAI-compatible server.

## Basic Example

```baml
client<llm> MyClient {
    provider "openai-generic"
    options {
        base_url "http://localhost:1234/v1"
        model "TheBloke/phi-2-GGUF"
    }
}
```

## BAML-Specific Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `base_url` | string | - | `http://localhost:1234/v1` |
| `model` | string | - | Model identifier |

See [LM Studio documentation](https://lmstudio.ai/docs/local-server#make-an-inferencing-request-using-openais-chat-completions-format) for more information.
