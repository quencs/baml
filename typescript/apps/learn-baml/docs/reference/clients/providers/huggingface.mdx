# Hugging Face

[Hugging Face](https://huggingface.co/) provides access to thousands of models via their Inference Endpoints.

## Basic Example

```baml
client<llm> MyClient {
    provider openai-generic
    options {
        base_url "https://api-inference.huggingface.co/v1"
        api_key env.HUGGINGFACE_API_KEY
        model "meta-llama/Llama-3-70B-Instruct"
    }
}
```

## BAML-Specific Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `base_url` | string | - | `https://api-inference.huggingface.co/v1` |
| `api_key` | string | `env.HUGGINGFACE_API_KEY` | API key |
| `model` | string | - | Model identifier |

See [Hugging Face Inference Endpoints](https://huggingface.co/docs/inference-endpoints/index) for more information.
