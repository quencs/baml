# Cerebras

[Cerebras](https://inference-docs.cerebras.ai/resources/openai) provides ultra-fast inference using their custom Wafer-Scale Engine hardware.

## Basic Example

```baml
client<llm> CerebrasLlama {
    provider "openai-generic"
    options {
        base_url "https://api.cerebras.ai/v1"
        api_key env.CEREBRAS_API_KEY
        model "llama-3.3-70b"
    }
}
```

## BAML-Specific Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `base_url` | string | - | `https://api.cerebras.ai/v1` |
| `api_key` | string | `env.CEREBRAS_API_KEY` | API key |
| `model` | string | - | Model identifier |

See [OpenAI Generic](/reference/clients/providers/openai-generic) for all supported options.
