# LiteLLM

[LiteLLM](https://www.litellm.ai/) provides a unified API proxy for 100+ LLM providers.

## Setup

1. Set up [LiteLLM Proxy server](https://docs.litellm.ai/docs/proxy/docker_quick_start#21-start-proxy)
2. Configure LiteLLM client in BAML
3. Use in BAML functions

## Basic Example

```baml
client<llm> MyClient {
    provider "openai-generic"
    options {
        base_url "http://0.0.0.0:4000"
        api_key env.LITELLM_API_KEY
        model "gpt-4o"
    }
}
```

## BAML-Specific Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `base_url` | string | - | LiteLLM proxy URL |
| `api_key` | string | `env.LITELLM_API_KEY` | API key |
| `model` | string | - | Model identifier |

See [OpenAI Generic](/reference/clients/providers/openai-generic) for all supported options.
